# SwiftBasics

program to sum of the numbers in an array.

var numbersArray = [35,45,55,65];

var total = 0
for i in numbersArray {
  total = total + i
}
print("total..\(total)")

program to multiply of the numbers in an array.

var numbersArray = [35,45,55,65];

var total = numbersArray[0]
for i in numbersArray {
  total = total * i
}
print("total..\(total)")
